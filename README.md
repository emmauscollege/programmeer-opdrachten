# Kleine programmeeropdrachten

Dit repository bevat 15 kleine programmeeropdrachten om je te leren kleine problemen in code op te lossen. Ga als volgt te werk:

1. Bij het open van je Codespace wordt een webpagina geopend die voor iedere opdracht aangeeft of de code doet wat er verwacht wordt. Wil je deze webpagina handmatig openen, ga naar 'Poorten', ga met de je muis naar de link naast '3000' en klik op het werelbolletje.
2.  Ga in Codespaces naar `opdracht-1.js` en pas daar de code zo aan zodat deze voldoet aan de opdracht. Sla de wijzigingen op.
3.  Herlaad de webpagina met feedback. Als je het goed hebt gedaan, staan alle vinkjes bij opdracht 1 nu op groen. Zo niet, ga terug naar stap 2.
4.  Herhaal stap 2 en 3 voor de volgende opdrachten.
5.  Stop je (tussendoor)? Sla alles op en commit + push de wijzigingen.
